# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/abhinavgupta30/dev-detective.git
```

Go into the project

```bash
cd dev-detective
```

Install dependencies

```bash
npm install
```

Run locally

```bash
npm run dev
```

Build production version

```bash
npm run build
```
